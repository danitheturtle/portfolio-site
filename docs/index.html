My Principles

Derived from years of on-the-job experience, combined with my hyper-effective fusion of design and engineering expertise.

Don't Re-invent

Someone has already done it better, faster and cleaner. Sharing knowledge is a human superpower; use it.

Invert Control

Devs are users, so code needs excellent UX. Future proof code with dependency injection, robust configuration and modularity.

Focus on the User

No design survives user testing. Talk to users early and talk to them often.

Testing is Foundational

Test Driven Development combined with User Testing is the only way to assure sustainable project growth.

UX from Accessibility

Good, clean UX flows directly from good accessibility. Following a11y best practices often spotlights UI/UX issues.

Expect Change

Be it stakeholder requests, user feedback or architecture shifts: anticipate change and operate with systemic flexibility.
